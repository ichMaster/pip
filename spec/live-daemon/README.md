# spec/live-daemon

Design docs for the **Live Daemon** track (**v1**, phases `v1.0`–`v1.5`) —
turning pip into a resident `pip-server` daemon + a live `pip-client`, with an
async mind, canon LLM chat, and session memory. Plan only; **not implemented
yet**.

This track is the **foundation** for the front-desk vision (**v2**) in
[`../vision/pip-frontdesk-spec.md`](../vision/pip-frontdesk-spec.md) (the final
target state), kept as a standalone, educational-first deliverable.

**Start here:** Architecture & Mission for the *why* and the *shape* of v1; the
global [ROADMAP](../ROADMAP.md) for the phased plan (`v1.0` → `v1.5`). The five
requested changes are delivered by the end of **`v1.3`**; `v1.4`–`v1.5` harden it
and bridge to the v2 vision.
