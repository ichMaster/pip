# pip

Чат-тамагочі. Маленьке створіння живе у твоєму терміналі: має потреби (голод,
енергія, настрій), що спадають у реальному часі, саме подає голос про те, як
почувається, і ти з ним і розмовляєш, і дбаєш про нього — усе в одному чаті.
Голос — модель Anthropic (Haiku); без API-ключа вмикаються прості templated-репліки,
і воно все одно живе.

Це спрощення світових симуляцій (mote/Clay) до самої суті: **прибрано світ,
лишилося тіло + потреби + LLM як голос і дрібні рішення.** Той самий принцип двох
годинників — тіло-потреби тікають самі (чиста математика, завжди), а розум (LLM)
запитують лише коли є що сказати.

## Особливості

- **Час спливає по-справжньому.** Відійдеш — повернешся до зголоднілої pip. Голод,
  енергія й настрій спадають з реальним часом, навіть поки ти не пишеш.
- **Статус — стабільна картинка.** Мордочка й смужки — це один блок унизу, що
  оновлюється *на місці* (а не друкується заново щоразу). Смужки повзуть у
  реальному часі прямо на очах; розмова прокручується над ними.
- **Сама говорить про потреби.** Коли потреба падає низько, pip подає голос без
  запиту: «...i'm getting really hungry».
- **Чат і догляд разом.** Звичайний текст — це розмова (pip відповідає в характері,
  згадуючи, як почувається). Команди — це турбота.
- **Настрій реагує на тебе.** Доброта і гра піднімають настрій; коли ігнорують —
  він тихо сповзає. LLM повертає не лише репліку, а й дрібний зсув настрою.

## Команди в чаті

| команда | дія |
|---|---|
| `/feed` | погодувати (піднімає ситість) |
| `/play` | погратися (піднімає настрій, виснажує) |
| `/sleep` | дати поспати (відновлює енергію) |
| `/status` | показати потреби |
| `/help` | список команд |
| `/quit` | вийти |

Будь-який інший текст — це просто розмова.

## Запуск

```
pip install anthropic
cp .env.example .env        # і впиши свій ANTHROPIC_API_KEY у .env
python pip.py
```

Ключ можна задати або у `.env` (git його ігнорує), або через `export
ANTHROPIC_API_KEY=...` — змінна з оточення має перевагу. Налаштування LLM
(`PIP_MODEL`, `PIP_MAX_TOKENS`) теж живуть у `.env`.

Варіанти:

```
python pip.py --name moss      # інше ім'я
python pip.py --speed 2        # потреби спадають удвічі швидше
python pip.py --rule           # без API, лише templated-голос
```

На старті pip вітає тебе **обрамленим банером** у дві колонки: ліворуч — панель
`pip` із мордочкою та смужками потреб (тобто статус — **перед чатом**), праворуч —
команди. Банер підлаштовується під ширину терміналу:

```
  ╭─ pip ──────────────────────╮  ╭─ commands ───────────────────────╮
  │ (✶‿✶)                      │  │ /feed /play /sleep   care for me │
  │ hunger ###############---- │  │ /status /help /quit  info        │
  │ energy #################-- │  │ anything else        just talk   │
  │ mood   #############------ │  │                                  │
  ╰────────────────────────────╯  ╰──────────────────────────────────╯
```

Далі розмова просто прокручується у звичайному терміналі. Статус **не висить**
унизу — це знімок: щоб подивитися поточний стан, набери `/status`, а після
догляду (`/feed` / `/play` / `/sleep`) панель `pip` друкується заново сама:

```
  you> i missed you today
  pip: i missed you too. though i'm getting a little hungry...
  you> /feed
  pip: mmm, thank you. that's better.
  ╭─ pip ──────────────────────╮
  │ (✦‿✦)                      │
  │ hunger ################---- │
  │ energy ###############---- │
  │ mood   ##################-- │
  ╰────────────────────────────╯
  you>
```

Це звичайний термінал зі скролбеком, без «захоплення екрана». (Живий статус, що
оновлюється сам, повернеться з повноекранним клієнтом демона у v1.1.)

Мордочки — у мінімалістичному, «сяйливому» стилі (близькому до Claude Code), і
колір рамки міняється зі станом: щаслива `(✦‿✦)`, спокійна `(✶‿✶)`, звичайна
`(◦‿◦)`, голодна `(✺o✺)`, втомлена `(-_-)`, сумна `(⌢_⌢)`. Є ще окрема мордочка
«думає» `(✻⋅✻)`, яку показуватимемо, поки pip готує відповідь (вмикається у v0.4).
Усі гліфи — одноширинні (перевіряються програмно), щоб рамка не «попливла».

## Як це влаштовано

- **Тіло (швидкий тік):** три потреби спадають із реальним часом. Чиста математика.
- **Догляд (ти):** `/feed` / `/play` / `/sleep` міняють числа.
- **Голос + рішення (LLM):** на твою репліку чи дію pip відповідає в характері,
  згадуючи свій стан, і повертає дрібний зсув настрою. Спонтанні нагадування про
  потреби — templated (миттєві й безкоштовні), щоб фоновий голос нічого не коштував.

Без `ANTHROPIC_API_KEY` чи без пакета `anthropic` — тихий фолбек на templated-голос.

Введення використовує `select` на stdin — це Unix (macOS/Linux). Жодних
зовнішніх залежностей, крім опційного `anthropic`.

## Структура коду

Це навчальний проєкт. Код розкладено на пакет `creature/`, де кожен модуль —
одна ідея (`pip.py` лишається тонким лаунчером):

| модуль | роль |
|---|---|
| `creature/needs.py` | **ТІЛО** — потреби й їхній спад у реальному часі (чиста математика) |
| `creature/brain.py` | **РОЗУМ** — templated- і LLM-голос за єдиним інтерфейсом |
| `creature/config.py` | конфіг LLM (модель, токени, ключ), що читається з `.env` |
| `creature/render.py` | мордочка та смужки (чисті функції від стану) |
| `creature/pet.py` | оркестрація — де тіло й розум зустрічаються |
| `creature/chat.py` | цикл реального часу |
| `creature/cli.py` | прапорці та запуск |

Запустити можна як `python pip.py`, так і `python -m creature`. Повний розбір
архітектури — у [GUIDE_uk.md](GUIDE_uk.md).

## Куди далі

- Пам'ять: pip пам'ятає попередні розмови між сесіями (зберегти стан + журнал на диск).
- Двошаровий голос: templated для рутинних реакцій, LLM для справжньої розмови.
- Свій канон: підстав власну особистість у SYSTEM-промпт.
- Стани, що тривають: хвороба від занедбаності, сон як окремий режим із плином часу.
